This is the MATLAB code for the numerical results contained in the work "Price-volume agreements: a one principal/two agents model"
by Francesco Biancalani, Giorgio Gnecco, and Massimo Riccaboni. The main file is "simulations_price_volume_agreement.m". To run the
code, the symbolic and optimization MATLAB toolboxes are required. A description of the code is provided in the work "Price-volume
agreements: a one principal/two agents model".
